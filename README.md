# LiquidShopGUI

A modern Paper 1.21+ economy and shop plugin — **under development**.

## Commands

| Command | Description |
|---------|-------------|
| `/balance` `/bal` | Check your balance |
| `/balancetop` `/baltop` | Top 10 richest players |
| `/pay <player> <amount>` | Send money to another player |
| `/shop` | Open the shop GUI |
| `/worth [item]` | Check an item's value |
| `/sell [hand\|all\|blocks]` | Sell items for cash |
| `/eco give\|take\|set\|reset <player> [amount]` | Admin economy management |
| `/eco reload` | Reload configuration |
| `/setworth <item> <price>` | Set an item's value |

## Status

Early development. Shop GUI, persistence (SQLite), and scoreboard coming soon.
